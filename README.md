### DAY-1 Practice

# Project: Student Records Management System

## Objective

Create a Student Records Management System using Microsoft Excel by applying the concepts learned in the Excel Fundamentals course.

### Dataset

Use the provided dataset containing:

* S.NO
* First Name
* Last Name
* DOB
* Telugu
* Hindi
* Science
* English

---

## Task 1: Prepare the Worksheet

* Create a title: **STUDENT RECORDS**
* Format the table professionally.
* Apply borders, header color, bold text, and center alignment where appropriate.
* Format the DOB column as a Date.

---

## Task 2: Perform Calculations

Create the following new columns:

### Total Marks

Calculate the total marks obtained by each student in all four subjects.

### Average Marks

Calculate the average marks of each student.

### Maximum Marks

Find the highest mark scored by each student among the four subjects.

### Minimum Marks

Find the lowest mark scored by each student among the four subjects.

### Full Name

Combine the First Name and Last Name into a single column named **Full Name**.

---

## Task 3: Sort the Data

Perform the following sorting operations:

* Sort students by **Average Marks** in descending order (Highest to Lowest).
* Sort students alphabetically by **First Name**.

---

## Task 4: Filter the Data

Apply filters to display:

* Students whose Average Marks are greater than 85.
* Students whose Science marks are less than 70.

---

## Task 5: Apply Conditional Formatting

Apply formatting rules:

* Highlight subject marks below 35 using a red fill color.
* Highlight Average Marks greater than 90 using a green fill color.

---

## Task 6: Create Charts

Create the following charts:

### Column Chart

Display each student's **Full Name** against their **Total Marks**.

### Pie Chart

Display the distribution of Total Marks for all students.
(Alternatively, create a pie chart showing the total marks obtained in each subject.)

---

## Task 7: Create a Pivot Table

Create a Pivot Table that shows:

* Average Marks grouped by Last Name.
* Total Marks grouped by Last Name.

---

## Bonus Task

If you have learned the IF() function:

### Result

Display:

* Pass if the student scores 35 or above in every subject.
* Fail if the student scores below 35 in any subject.

### Grade

Assign grades based on the Average Marks.

Example:

* A : 90 and above
* B : 80–89
* C : 70–79
* D : 60–69
* F : Below 60

---
