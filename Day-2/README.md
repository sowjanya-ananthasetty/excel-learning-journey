# Inventory Management System (Microsoft Excel)

## 📌 Project Overview

Built an **Inventory Management System** using Microsoft Excel to practice data management, formulas, lookup functions, data visualization, and Pivot Tables.

---

## 📚 Concepts Practiced

### Excel Functions
- IF()
- Nested IF()
- VLOOKUP()
- SUM()
- COUNTIF()

### Inventory Calculations
- Stock Status (In Stock / Low Stock)
- Stock Alerts (Out of Stock / Order Now / Available)
- Inventory Value Calculation

### Data Management
- Excel Tables
- Sorting
- Filtering
- Conditional Formatting

### Data Visualization
- Column Chart
- Pie Chart
- Bar Chart

### Data Analysis
- Pivot Tables
- Dashboard (Basic KPIs)

---

## 🛠 Features Implemented

- Product Inventory Database
- Price List
- Order Sheet using VLOOKUP
- Stock Status Calculation
- Stock Alert System
- Inventory Value Calculation
- Conditional Formatting for Stock Levels
- Product Sorting
- Product Filtering
- Charts for Inventory Analysis
- Pivot Tables for Summary Reports
- Inventory Dashboard

---

## 📊 Dashboard KPIs

- Total Products
- Total Inventory Value
- Low Stock Products

## 🎯 Skills Gained

- Applied logical functions using IF() and Nested IF()
- Retrieved data using VLOOKUP()
